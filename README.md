# 🎓 Grade Calculator

This project was created as a **learning project** while studying Python and exploring how to build interactive web applications.

## ⚙️ Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Move into the project folder:

```bash
cd Grade-Calculator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
